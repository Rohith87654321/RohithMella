AI-Driven Financial Forecasting System

This is a simple finance forecasting AI model. The primary goal of this project is to predict future prices of stocks or currencies using historical price data. The predictions are made for different time frames: next 24 hours, 7 days, 1 month, and 1 year.

Overview:

  • This is a simple finance forecasting AI model that predicts future stock or currency prices using historical data.
  • The model forecasts prices for the next 24 hours, 7 days, 1 month, and 1 year.
  • It uses data from Yahoo Finance and applies a Linear Regression model.

Technologies:

  • Python: Main programming language used for scripting and data manipulation.
  • yfinance: Library used to fetch historical financial data.
  • scikit-learn: Used for implementing the Linear Regression model.

Methodology:

  • Data Collection: Fetch historical data for the given stock or currency symbol from Yahoo Finance.
  • Model Training: Train a Linear Regression model on the historical data.
  • Prediction: Forecast future prices for specified time frames using the trained model.

Key Contributions:

  • Developed a tool for simple and quick financial forecasting using readily available data.
  • Implemented a user-friendly script that interacts with the user to input stock symbols and view predictions.
  • Enhanced data visualization with ASCII art and detailed informational outputs.

Outcome:

  • Provided an easy-to-use financial forecasting tool that can predict future prices with basic accuracy.
  • Highlighted the potential of using machine learning models for financial predictions.
  • Created a foundation that can be expanded with more sophisticated models and additional features for better accuracy.
