Customer Sentiment Analysis for E-commerce

This project is a Flask-based web application for sentiment analysis of product reviews. It involves creating a machine learning model to predict the sentiment of a given product review (positive or negative) and integrating this model into a Flask web application for real-time sentiment analysis.

Overview:

  • Developed a Flask-based web application for real-time sentiment analysis of product reviews.
  • Integrated a machine learning model to predict the sentiment (positive or negative) of user-submitted reviews.
  • Provided a user-friendly interface for entering and analyzing reviews.

Technologies:

  • Programming Languages: Python, HTML, CSS
  • Libraries/Frameworks: Flask, Scikit-learn, Pandas, Jinja2
  • Tools: Jupyter Notebook, Joblib, TF-IDF Vectorizer

Methodology:

  • Data Preprocessing: Cleaned and vectorized text data using TF-IDF.
  • Model Training: Trained a Logistic Regression model for sentiment classification.
  • Web Integration: Built and deployed the model within a Flask web application.

Key Contributions:

  • Implemented efficient text preprocessing and vectorization techniques.
  • Developed a robust machine learning model achieving high prediction accuracy.
  • Created an intuitive web interface for seamless user interaction and real-time sentiment analysis.

Outcome:

  • Achieved an accurate and scalable sentiment analysis solution.
  • Enabled users to easily assess the sentiment of product reviews.
  • Demonstrated the ability to integrate machine learning models with web applications for practical use cases.
