Business Process Improvement for Retail Store

This project involves developing automated workflows using Microsoft Power Automate to enhance financial process efficiency and accuracy. It specifically focuses on normalizing and validating cheque numbers extracted from financial documents, ensuring seamless integration into broader automation frameworks.

Overview:

  • Developed automated workflows using Microsoft Power Automate for financial process optimization.
  • Implemented a flow to validate and normalize cheque numbers extracted from documents.
  • Enhanced data integrity and compliance by ensuring accurate formatting before integration into downstream systems.

Technologies:

  • Programming Languages: Microsoft Power Automate (no-code platform),SQL
  • Tools: Microsoft Power Automate, Microsoft Office Suite (Excel), Visio
  • Integration Tools: OCR tools for data extraction, other RPA tools for orchestration (e.g., Blue Prism)

Methodology:

  • Data Extraction: Utilized OCR tools to extract cheque numbers from financial documents.
