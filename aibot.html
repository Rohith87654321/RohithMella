Web Crawler and AI Chatbot

The web crawler's purpose in this project is to collect relevant information from various web pages about a specific subject and build a knowledge base that the chatbot can use to provide informed responses.The chatbot uses the knowledge base created by the web crawler to provide relevant information to users about Leonardo DiCaprio.

Overview:

  • Objective: Develop a web crawler and chatbot to gather and provide information about Leonardo DiCaprio.
  • Components: Web crawler for data extraction, chatbot for user interaction.
  • Scope: Collect, process, and use web data to answer user queries.

Technologies:

  • Programming Languages: Python
  • Libraries: NLTK, BeautifulSoup, Requests, Scikit-learn
  • Tools: TF-IDF vectorizer, Cosine similarity

Methodology:

  • Data Collection: Use web crawler to extract text from web pages.
  • Data Processing: Clean and tokenize text data, extract key terms using TF-IDF.
  • Interaction: Implement a chatbot that matches user input with knowledge base terms.

Key Contributions:

  • Web Crawler: Efficiently gathered and processed large amounts of web data.
  • Knowledge Base: Created a comprehensive database of relevant information using NLP techniques.
  • Chatbot: Developed an interactive system capable of understanding and responding to user queries.

Outcome:

  • Data Coverage: Successfully crawled multiple sources, creating a rich knowledge base.
  • User Interaction: Provided accurate and relevant information through the chatbot.
  • Scalability: Demonstrated a scalable approach to combining web crawling and chatbot technologies.
