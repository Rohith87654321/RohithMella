Customer Satisfaction Analysis for Financial Services

This project focuses on analyzing customer satisfaction for CIH Bank by utilizing sentiment analysis on customer feedback collected from Google Maps. It involves developing a robust machine learning model using BERT to classify sentiments and integrating it into an ETL pipeline for structured data management and visualization through Power BI.

Overview:

  • Developed a comprehensive ETL pipeline to extract customer feedback data from Google Maps using the Apify API.
  • Employed a BERT-based machine learning model for real-time sentiment analysis of customer reviews.
  • Integrated data visualization tools (Power BI) to present customer satisfaction metrics and trends effectively.

Technologies:

  • Programming Languages: Python, SQL, R
  • Libraries/Frameworks: BERT, Apache Airflow, Pandas, NumPy.
  • Tools: PostgreSQL, Power BI, SPSS

Methodology:

  • Data Extraction: Utilized Apify for scraping customer reviews from Google Maps, ensuring diverse data collection.
  • Data Processing: Cleaned and transformed data using Python scripts before loading it into PostgreSQL for storage.
  • Sentiment Analysis: Fine-tuned the BERT model for sentiment classification and deployed it to analyze customer feedback in real-time.

Key Contributions:

  • Implemented an efficient ETL process to streamline data collection and analysis.
  • Developed a robust sentiment analysis model with BERT, achieving high accuracy in sentiment classification.
  • Created interactive Power BI dashboards for dynamic visualization of customer satisfaction trends across different branches.

Outcome:

  • Achieved actionable insights into customer satisfaction levels, facilitating targeted improvements in service quality.
  • Enabled CIH Bank to monitor customer sentiment in real-time, enhancing decision-making processes.
  • Demonstrated the effectiveness of integrating machine learning models with data visualization tools for impactful business intelligence applications.
