Emotion Detection using LLM

The main goal of this project is to perform sentiment analysis on Twitter data, classifying tweets into multiple emotion categories. The project utilizes several transformer models to achieve multi-label classification, aiming to compare their performance and effectiveness.

Overview:

  • Objective: Perform sentiment analysis on Twitter data, classifying tweets into 11 emotions.
  • Scope: Evaluate multiple transformer models for multi-label classification.

Technologies:

  • Libraries: Used transformers, datasets, torch, and sklearn.
  • Models: Implemented DistilBERT, DistilRoBERTa, FLanT5, LLAMA, Qwen, and Gemma.
  • Tools: Jupyter Notebooks, Google Colab, and Weights and Biases.

Methodology:

  • Preprocessing: Custom tokenization and data preparation for each model.
  • Training: Configured and trained models using TrainingArguments and Trainer API.
  • Evaluation: Computed accuracy and F1 score, visualized with matplotlib.

Key Contributions:

  • Comparison: Detailed study of multiple transformer models.
  • Preprocessing: Custom functions for tailored data handling.
  • Tracking: Used Weights and Biases for systematic experiment tracking

Outcome:

  • Performance: Identified strengths and weaknesses of different models.
  • Insights: Provided recommendations for effective sentiment analysis.
